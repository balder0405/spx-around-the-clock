# Markets Around the Clock 🌐

A public, no-forecast 24/7 view of **SPX & NDX** plus the single stocks that actually trade
around the clock — every number a real print from a named venue, or a clearly-marked estimate.

- **Indices (SPX, NDX):** cash index in RTH · CME futures (ES/NQ − basis) overnight · Hyperliquid
  US500/USTECH perp on weekends.
- **Single stocks (13):** US consolidated print 4 AM–8 PM ET · **Hyperliquid single-stock perp mark**
  (builder-dex perps) overnight & weekends — marks validated to track spot within ~1%.
- Only tickers with a genuine 24/7 venue are shown. No fabricated overnight quotes.

Run: `streamlit run streamlit_app.py`
